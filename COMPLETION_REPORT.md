## 🚀 部署方案

### 方案 1：本地预览
```bash
npx -y http-server . -p 8000
# 访问 http://localhost:8000
```

### 方案 2：GitHub Pages（推荐）
1. 创建 GitHub 仓库
2. 推送所有文件到 main 分支
3. 启用 GitHub Pages（Settings > Pages）
4. 网站将在 `https://username.github.io/repo` 发布

### 方案 3：传统虚拟主机
- 上传所有文件到 Web 服务器
- 设置首页为 `index.html`
- 配置 404 重定向（单页应用）
